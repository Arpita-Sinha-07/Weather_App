# 🎯 About

The **Weather App** is a modern web application developed using **HTML, CSS, and JavaScript** that fetches real-time weather data and presents it through a clean, responsive, and visually appealing interface.

This project demonstrates:

* API Integration
* Asynchronous JavaScript
* DOM Manipulation
* Responsive Web Design
* Error Handling
* User Experience Design

The application enables users to instantly search weather conditions for any city and receive detailed weather insights.

# ⚙️ How It Works

### Step 1

User enters a city name.

### Step 2

JavaScript sends a request to the weather API.

### Step 3

Weather data is retrieved in JSON format.

### Step 4

Application extracts:

* Temperature
* Humidity
* Wind Speed
* Weather Condition

### Step 5

Results are dynamically displayed on the webpage.
